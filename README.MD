 ## 📁 Como baixar o projeto

```bash
# clonar repositório
$ git clone https://github.com/wagneer-nascimento/api-clientes.git

# Configurar o banco local ormconfig.json
# Criar tabelas no banco
$ yarn typeorm migration:run

# Entrar no diretório
$ cd api-clientes

# Instalar as dependencias
$ yarn

# inicar o projeto
$ yarn dev


```
